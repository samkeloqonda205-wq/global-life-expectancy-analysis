# Global Life Expectancy Data Analysis

## Project Overview
This project explores global life expectancy trends using Microsoft Excel. 
The dataset was analyzed to identify patterns related to GDP, economic status, and life expectancy outcomes.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Dashboard Visualization

## Analysis Included
- Country vs Life Expectancy
- Life Expectancy Trends by Year
- GDP vs Life Expectancy
- Economic Status Comparison

## Dashboard
An Excel dashboard was created to visualize key insights and make the data easier to interpret.

## Objective
The goal of this project was to strengthen my practical data analysis and data storytelling skills as I continue growing in the analytics and tech space.
